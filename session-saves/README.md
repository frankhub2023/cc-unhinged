# Session Saves

Conversation checkpoints from Claude Code sessions on `cc-unhinged`.

## Naming convention
`YYYY-MM-DD-slug.md` — one file per session or major topic.

## Sections
- **What we did** — actions taken, code written, problems solved
- **Decisions made** — choices that affect future work
- **Open threads** — things mentioned but not finished
- **Next up** — intended next steps

## How to trigger a save
Say `save:"slug"` in any message — Claude will write the file, commit, and push.
